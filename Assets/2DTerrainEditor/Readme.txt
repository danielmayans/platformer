2D Terrain Editor v1.1

---

This package contains a simple tool for Unity that allows you to create, edit or randomly generate 2D terrain. Create terrain for your 2D game with one click or generate it in runtime. Feels great on mobile and takes only 1-2 drawcalls. Perfect solution for 2D racing games with Physics2D components.

Check out the documentation for more information.
